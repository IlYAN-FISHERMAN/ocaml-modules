# OCaml Modules

This repository contains my work on the **OCaml modules** from the
**42 School Piscine OCaml**.

## About the project

The goal of this project is to practice core functional programming
concepts in OCaml through progressive modules and exercises, including:

- OCaml syntax and types
- pattern matching and recursion
- lists and higher-order functions
- modules and abstract data types

It follows the learning approach used during the 42 Piscine:
small hands-on exercises, regular practice, and incremental difficulty.
